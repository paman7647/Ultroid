# MVP Roadmap (2-3 Milestones)

## Milestone 1 (Core)
- Auth and session management
- Room/DM creation
- Realtime messaging and typing
- Mobile-first chat UI shell

## Milestone 2 (Security + Upload)
- Secure attachment flow with scan pipeline
- Read receipts and presence reliability
- Profile/security settings
- Observability dashboards and alerts

## Milestone 3 (Cutover)
- Flask data migration + dual write
- Load/performance and accessibility verification
- Production deployment and rollback rehearsals
