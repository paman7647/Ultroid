# Information Architecture and User Flows

## Top-level IA
- Public
  - Landing
  - Login
  - Register
  - Forgot Password
- Authenticated App
  - Inbox (room list + DMs)
  - Room Detail
  - Contacts
  - Settings
    - Profile
    - Security
    - Devices
    - Notifications

## Core Flows
1. Onboarding
- Register -> Email verify -> Login -> Seed profile -> Create or join first room

2. Contact/DM
- Search user -> Open user profile card -> Start DM -> Send message -> Presence and read receipts visible

3. Room Collaboration
- Create group -> Invite members -> Join room -> Timeline + attachments -> Typing + read receipts

4. Security/Session
- Open security settings -> Review active devices -> Revoke a device -> Rotate password -> Reauthenticate
