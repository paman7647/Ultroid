# Design System

## Principles
- Mobile-first layout and interaction targets
- WCAG 2.2 AA contrast and keyboard navigation
- Low-end device performance budget

## Tokens
- Primary: `#0B57D0`
- Background Light: `#F7FAFC`
- Background Dark: `#0E1726`
- Success: `#0E9F6E`
- Warning: `#B7791F`
- Danger: `#C53030`

## Typography
- Heading: Space Grotesk
- Body: Source Sans 3
- Mono: JetBrains Mono

## Component Baseline
- Buttons: primary / outline / ghost
- Cards: rounded + border + soft elevation
- Chat: room list row, bubble, composer, typing indicator
- Settings: form sections and security action cards
